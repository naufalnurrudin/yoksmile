Answer

Website adalah kumpulan halaman web yang terkait dan dapat diakses melalui internet. Cara kerja website dimulai dengan pengguna mengakses website melalui browser, kemudian browser akan mengirimkan permintaan ke web server untuk mengambil informasi yang dibutuhkan. Web server akan mengirimkan informasi tersebut kembali ke browser dan browser akan menampilkan informasi tersebut dalam bentuk halaman web.

Domain adalah alamat unik yang digunakan untuk mengakses website. Hosting adalah tempat penyimpanan data website dan aplikasi yang dapat diakses melalui internet. Hubungan antara web server, domain, dan hosting adalah bahwa domain digunakan untuk mengakses website yang disimpan di hosting, dan web server adalah tempat di mana hosting disimpan dan mengirimkan informasi yang diminta oleh browser. Dalam hal ini, web server dan hosting dapat dianggap sebagai hal yang sama.
